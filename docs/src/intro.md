Welcome to the official documentation for **SMOC**. 

SMOC is a symbolic mathematics engine designed to process, simplify, and solve complex mathematical equations with absolute precision. Unlike standard calculators that rely on floating-point numbers (and lose precision), SMOC translates everything into an **Abstract Syntax Tree (AST)** and manipulates it algebraically.
